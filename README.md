# BlackScreen

Small .NET 10 Windows Forms utility that covers a selected monitor with a fullscreen black overlay.

## Features
- Choose which connected monitor to cover from the main window.
- Puts a borderless, topmost, black window on the selected screen.
- Close the black overlay by double-clicking anywhere on it.

## Usage
1. Run the application.
2. Select a monitor from the dropdown.
3. Click the button to make the selected screen black.
4. Double-click the black screen to restore the application.

## Build
- Targets .NET 10.
